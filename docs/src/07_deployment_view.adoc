[[section-deployment-view]]
== Verteilungssicht

Die Applikation wird mittels Docker Compose als vier Container betrieben. Ein separater Nginx Reverse Proxy ist der einzige Einstiegspunkt für den Browser und routet die Anfragen anhand der Domäne an die richtigen internen Container weiter.

image::deployment-view.png[Deployment-Diagramm, scale=false]

[cols="1,1,2", options="header"]
|===
| Container | Image | Beschreibung
| **nginx** | Nginx (Reverse Proxy) | Einziger Einstiegspunkt (Port 80/443). Routet anhand der Domäne an Client oder Server.
| **client** | client | Liefert die gebaute Angular-Applikation als statische Dateien aus (Port 4200).
| **server** | server | REST API mit Prisma ORM für die Technologie-Verwaltung (Port 3000).
| **db** | PostgreSQL 15 | Relationale Datenbank zur Persistierung der Technologiedaten (Port 5432).
|===
